== Description ==

*** This plugin has been integrated into the plugin of: <a href="https://wordpress.org/plugins/bsk-gravityforms-blacklist/">BSK Forms Blacklist</a>

*** Just install the plugin of BSK Forms Blacklist, you data and settings in this plugin will be transfered automatically.

The plugin help you avoid spam submissions from Contact Form 7. You may set it to use blacklist, whitelist, ip address or email to validate visitor’s input and only allow valid  entry submitted. It support validate multiple fields.

It is easy to use. You may create your blacklist, whitelist, or email database first, then apply them to one or multiple fields of your form. 

It also supports preventing submitting or disable notifications ( skip mails ).

When add email address it support domain name such as *@hotmail.com.

== Installation ==

It’s a standard WordPress plugin and you may install it from WordPress’ plugin Dashboard. After activate the plugin then you can create blacklist or whitelist or email list.

After it activated in your WordPress you'll see a submenu item of "Blacklist" under WordPress --> Dashboard --> Contact. 

There's a new tab named "Blacklist Settings" to let you apply blacklist items to form fields.
